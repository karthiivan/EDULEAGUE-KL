## ğŸ“š Feature Walkthrough

### 1. Dashboard
- **Overview**: See your personalized learning path, recent achievements, and recommended problems
- **Quick Stats**: View problems solved, current streak, total XP, and badges earned
- **Quick Actions**: Jump to coding rooms, find mentors, or start mock interviews

### 2. Coding Practice Hub
- **Browse Problems**: Filter by difficulty (Easy/Medium/Hard), topics, or companies
- **Solve Problems**: Click any problem to open the code editor
- **Code Editor**: Write code in JavaScript, Python, Java, or C++
- **Run & Submit**: Test your code and submit to earn XP
- **Rooms**: Create or join collaborative coding rooms with room codes

### 3. Learn Skills
- **5 Skill Tracks**: DSA, Competitive Programming, SQL, Frontend, Backend
- **Roadmaps**: Follow structured learning paths for each track
- **Quizzes**: Take 5-question quizzes to test your knowledge (+15 XP)
- **Resources**: Access curated learning materials

### 4. Peer Learning
- **Find Mentors**: Browse mentors by topic, rating, and availability
- **Filter**: Search by specific skills or topics
- **Schedule Sessions**: Book 1-on-1 mentorship sessions
- **Become a Mentor**: Apply to help fellow students

### 5. Resume Builder
- **Choose Template**: Select from 5 ATS-friendly templates
- **Auto-Fill**: Your profile data is automatically populated
- **ATS Checker**: Check your resume's ATS score (out of 100)
- **Download**: Export your resume as PDF

### 6. Projects
- **Browse Projects**: View all active project ideas
- **Filter by Category**: Web Dev, AI/ML, IoT, Mobile Apps, Blockchain
- **Join Teams**: Request to join projects that match your skills
- **Post Project**: Create your own project and find teammates

### 7. Interview Prep
- **Mock Interviews**: Schedule peer mock interview sessions
- **Company Questions**: Browse interview questions by company
- **Filter**: Search questions by Google, Amazon, Microsoft, etc.
- **Interview Tips**: Read experiences shared by alumni

### 8. Placements
- **Statistics**: View KL University placement stats
- **Company Visits**: See upcoming company visit dates
- **Filter by Branch**: View data for CSE, IT, ECE, or All
- **Selection Process**: Learn about each company's interview rounds
- **Eligibility**: Check CGPA and other criteria

### 9. Profile
- **Personal Info**: View your profile details and stats
- **Coding Profiles**: Display LeetCode, CodeChef, Codeforces, GitHub stats
- **Badges**: See all earned badges and achievements
- **Activity Heatmap**: View your coding activity calendar

### 10. Leaderboard
- **Rankings**: See your rank among all students
- **Filters**: Switch between Global, Year-wise, or Branch-wise
- **Top 3**: Featured display of top performers
- **Full List**: Complete leaderboard with XP, problems, and streaks

### 11. Teacher Portal (Teachers Only)
- **Student Tracking**: Monitor all student progress
- **Analytics**: View year-wise and branch-wise performance charts
- **Top Performers**: Identify high-achieving students
- **At-Risk Students**: Find students who need attention
- **Filters**: Filter by year and branch
- **Export**: Download reports as CSV/PDF

## ğŸ”§ Troubleshooting

### Login Issues
- Ensure you're using the correct email format
- Password is case-sensitive
- Use quick login buttons for demo

### Code Editor Not Loading
- Refresh the page
- Check browser console for errors
- Monaco editor requires modern browser

- Check browser localStorage is enabled
- Clear cache and try again

### XP Not Updating
- Refresh the page to see updated XP
- XP updates are saved to localStorage
